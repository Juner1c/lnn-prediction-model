# LNN Prediction Model Project

This project focuses on building a multi-modal weather forecasting system that predicts short-term temperature and heat-index values. The system leverages spatial relationships among multiple Automated Weather (AW) stations, utilizing a hybrid structure consisting of Liquid Neural Networks (LNN) for preprocessing/denoising and Graph Neural Networks (GNN) for spatial-temporal forecasting.
